# Информационная система автосалона

WinForms приложение на C#, реализующее базовую систему управления автомобилями в автосалоне. Позволяет просматривать, добавлять, редактировать и удалять автомобили, а также сортировать данные по различным полям.

## Функциональность

- Просмотр списка автомобилей с возможностью редактирования **непосредственно в таблице**
- **Добавление** новых автомобилей
- **Удаление** выбранных автомобилей
- **Сортировка** по любому столбцу (с поддержкой сортировки по возрастанию и убыванию)
- Отображение иконки сортировки в заголовке столбца (стрелки `↑` / `↓`)
- Фильтрация по любым из полей с возможностью выбрать несколько фильтров одновременно
- Поиск по любому из полей
- Поля автомобиля:
  - Бренд
  - Модель
  - Год выпуска
  - Наличие (в наличии / под заказ)
  - Количество в наличии
  - Комплектация
  - Цена
  - Дополнительные скидки (госпрограммы и т.д.)

## Технологии

- Язык: **C#**
- Платформа: **.NET Framework / .NET 10+**
- Графический интерфейс: **WinForms**
- Коллекция данных: `BindingList<T>` для автоматического обновления UI

## Структура проекта

- `Form1.cs` — основная логика формы
- `Form1.Designer.cs` — визуальный дизайн формы
- `Car.cs` — класс, описывающий автомобиль
- `Program.cs` — точка входа в приложение
